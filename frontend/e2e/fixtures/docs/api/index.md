# API Reference

Complete API reference documentation.

## Overview

The API provides the following functionality:

- RESTful endpoints
- JSON responses
- Authentication via API keys

## Endpoints

See the [Endpoints Reference](./endpoints.md) for details on all available endpoints.

## Authentication

All API requests require authentication. Include your API key in the header:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.example.com/v1/users
```
